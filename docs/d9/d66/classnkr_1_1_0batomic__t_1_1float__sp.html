Detailed Description

template<float_tr float_p>
class nkr::$atomic_t::float_sp< float_p >

A specialization for floats to be used atomically.

Provides an easy to use interface for atomically working with nkr::f32_t and if running in 64-bit mode, nkr::f64_t as well.
