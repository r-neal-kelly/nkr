Detailed Description

template<typename any_p>
class nkr::$maybe_t::$built_in_sp::any_sp< any_p >

The default nkr::maybe_t specialization for built-in types.

This works with nkr::std_bool_t and integers. Floating point numbers and pointers each have their own specialization of this template.
