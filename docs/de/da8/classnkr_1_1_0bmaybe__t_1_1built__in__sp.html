Detailed Description

template<built_in_tr built_in_p>
class nkr::$maybe_t::built_in_sp< built_in_p >

A nkr::maybe_t specialization for built-in types, such as booleans, integers, floating point numbers, pointers, etc.

Built-in types need a class wrapper to behave in the same manner that a user-defined type does. This allows us to define an implementation of nkr::none_i.
