Detailed Description

template<real_tr real_p>
class nkr::$atomic_t::real_sp< real_p >

A specialization for real numbers to be used atomically.

Provides an easy to use interface for atomically working with nkr::r32_t and if running in 64-bit mode, nkr::r64_t as well.
